Karatsuba vs Standard Integer Multiplication

Runtime Complexity Comparison

Standard O(n²)
Karatsuba O(n^1.585)

Standard Multiplication

O(n²)

Each digit multiplied with every other digit. For 50 digits: ~2,500 operations

Karatsuba Algorithm

O(n^log₂3) ≈ O(n^1.585)

Divide-and-conquer with 3 recursive calls. For 50 digits: ~400 operations

At n=50 digits, Karatsuba is approximately 6x faster than standard multiplication
